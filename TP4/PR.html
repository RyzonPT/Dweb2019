Abstract:

Este trabalho consistiu em fragmentar o ficheiro arq.xml em vários ficheiros (um ARQELEM por ficheiro) e desenvolver um servidor de ficheiros em node.js capaz de devolver cada um desses ficheiros xml fragmentado aquando da recessão de um pedido indicando o ficheiro a devolver. Para além disso esse ficheiro deve ser transformado com uma stylesheet, no servidor, antes de ser devolvido ao cliente. No final ainda foi utilizado html juntamente com javascript para gerar uma página de erro conveniente.

Deliverables:

  • PR.xml
  • PR.html
  • arq2html.xsl
  • servidor.js
  • xml-split.xsl
  • Ficheros ARQ fragmentados
