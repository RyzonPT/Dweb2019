Abstract:

Este trabalho prático consistiu em criar um sistema de gestão de tarefas onde fosse possivel, na mesma página web, ver uma lista de tarefas e adicionar tarefas a essa lista.

Para isso, foram definidas as rotas principais Post: "/tarefa" e GET: "/","/tarefas","/w3.css".

Para além disso ainda foi definido uma pagina de erro para qualquer outra entrada com um botão de redirecionamento para a página principal.

Finalmente, na pagina web desenvolvida foi adicionado um relógio extra para ajudar na seleção das datas limite
