Abstract:

Este trabalho prático consistiu em contruir querys simples de Mongo e criar uma aplicação WEB com as operações CRUD sobre a base de dados de Filmes disponibilizado pelo Professor.

Para isso, foi gerado um novo projeto com a ajuda das funcionalidades do node Express, npm e mongoose.

Esta aplicação contem todas as rotas de delete,get,post e put necessárias ao seu bom funcionamente. Para fazer os pedidos ao sistema foi usada a framework Axios.

Mais uma vez, a interface incide maioritáriamente apenas numa página web para diminuir a quantidade de pedidos ao servidor.

Para além disso, devido ao facto de o dataset a ser tratado ter volume considerável, foram experimentadas daus abordagens para melhorar a velocidade de load da página. A primeira abordágem foi gerar os dados de todos os filmes no front-end, isto é, criar todas as linhas da tabela com javascript na parte do cliente para diminuir a carga do servidor. A segunda foi utilizar o pug para gerar todas as linhas da tabela.

De facto, a primeira abordagem mostrou-se bem mais lenta do que a segunda pelo que se utilizou o pug para tratar do trabalho pesado.
