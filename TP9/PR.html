Abstract:

Este trabalho consistiu em desenvolver um programa semelhante ao mongoimport, isto é, criar uma base de dados Mongo através de um fichero em formato JSON.

Para isso, foi gerado um programa divido em 3 fases, validação de input, criação do schema da BD e carregamento dos dados.

A validação garante que o formato do input recebido é valido, por exemplo, se o ficheiro introduzido existe e se encontra em formato JSON.

Depois, com a ajuda das funcinalidades fornecidas pela biblioteca mongoose, o ficheiro json de input é analisado para se criar o schema dinámicamente.

Por fim, mais uma vez com a ajuda do mongoose são inseridos todos os dados, do ficheiro input, na base de dados mongo, já preparada para os receber.
