Abstract:

Este Teste consiste em dois exercícios distintos.

O primeiro exercício consiste em criar uma API de dados a partir de um ficheiro XML fornecido. Em primeiro lugar, foi necessário transformar o ficheiro XML em ficheiro JSON. Para isso foi usado um conversor online para agilziar o processo de transformação.

Depois foi utilizada uma base de dados Mongo para realizar a persistencia do sistema. Para importar o dataset foi utilizado o seguinte comando: mongoimport --db musicas --collection musicas --file arquivo-musica-digital.json --jsonArray

Por fim, foram criados os models e controllers necessários para criar as rotas pedidas no enunciado.

O segundo exercicio consiste em consumir uma API de dados pré-feita. Este tem como objectivo criar uma interface que apresente os dados da API permitindo a navegação sobre a estrutura dos mesmos.

Para além disso, foram resolvidas quatro questões. As responstas são as seguintes

Resposta 1: 546 entidades

Resposta 2: Administração Pública , Autoridades de supervisão e fiscalização

Resposta 3: codigo: 400.10.056, titulo: Registo nacional de condutores

Resposta 4: codigo: 750.20.601, titulo: Realização de atividades de formação e treino animal
